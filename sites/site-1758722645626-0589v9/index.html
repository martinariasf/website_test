Hilfe & Anleitung

1. Dateien hochladen

Wählen Sie zwei Excel-Dateien (.xlsx oder .xls) aus, die Sie vergleichen möchten. Beide Dateien müssen ausgewählt werden, bevor der Vergleich gestartet werden kann.

2. Vergleich starten

Klicken Sie auf "Dateien vergleichen" um den automatischen Vergleichsprozess zu starten. Das Tool analysiert jede Zelle und identifiziert Unterschiede.

3. Ergebnisse anzeigen

Unterschiede werden farblich hervorgehoben angezeigt. Sie können zwischen verschiedenen Ansichten wechseln und einen Bericht herunterladen.

Hinweis: Das Tool funktioniert komplett im Browser. Ihre Dateien werden nicht auf einen Server hochgeladen und bleiben auf Ihrem Computer.
